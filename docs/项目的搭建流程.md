# 项目的搭建

## 项目的开始

> yarn create vite XXX
> cd xxx
> yarn add vue-router@4 || npm install vue-router@4  //安装路由插件
> yarn add pinia || npm install pinia                //安装状态管理
> yarn && yarn dev

## 简单的配置

简单的配置一下`vite.config.js`启动IP显示和文件路径别名

```js
// vite.config.js
import { join } from "path";
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'

export default defineConfig({
  plugins: [vue()],
  // 本地服务
  server:{
    open:true,
    port:5173,
    host:"0.0.0.0"
  },
  resolve: {
    alias: {
      '@': join(__dirname, "src"),
    }
  }
})
```

## 配置 pinia

创建文件夹`stores`然后在里面编辑pinia的简易代码

```js
// stores/index.js
import { createPinia,defineStore } from "pinia";
export const Store = createPinia()
export const useStore = defineStore('main',{
    state:()=>({
        counter:0
    }),
    getters:{
        double:(state)=> state.counter + 1,
    },
    actions:{
        addNum(){
            this.counter++
        }
    }
})
```

```js
// main.js
import { createApp } from 'vue'
import './style.css'
import App from './App.vue'
import {Store} from '@/stores'
createApp(App).use(Store).mount('#app')
```

```js
// App.vue
<template>
  ...
  <span @click="addNum">{{counter}} </span>
  ...
</template>
<script setup>
...
import { storeToRefs } from 'pinia';
import { useStore } from '@/stores';
const Store = useStore()
const {counter} = storeToRefs(Store)

const addNum=()=>{
  Store.addNum()
}
</script>
...
```
